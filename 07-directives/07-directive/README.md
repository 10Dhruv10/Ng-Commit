# 07-directives

## What I Learned

Custom directives:

- **Attribute directive** - `appHighlight` with `@Input()`, host listeners, DOM manipulation
- **Structural directive** - `*appStructural` using `TemplateRef` & `ViewContainerRef`
- **Comparison** with `*ngIf` and `ngNonBindable`

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

## Generate directive

Run `ng g d highlight` to generate a new directive.
