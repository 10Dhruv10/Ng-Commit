# 013-reactive-form

## What I Learned

Reactive forms:

- **FormControl**, **FormGroup**, **FormArray**
- **FormBuilder** service for simplified syntax
- **Validators** (`required`, `minLength`)
- **Form state tracking** (dirty, touched, valid)
- **valueChanges** / **statusChanges** observables
- **.patchValue()** for partial updates

Proceed to `src/profile-editor/profile-editor.component.ts` and `profile-editor.component.html`
