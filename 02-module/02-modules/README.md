# Essentials

## What I Learned

Converting standalone to NgModule architecture:

- **@NgModule** with declarations, imports, exports
- **Feature modules** (`TasksModule`, `SharedModule`)
- **Module-based bootstrapping**
- **Lazy loading preparation**

## How to Use

Start by exploring these key files:
- `src/app/app.module.ts` - Main module configuration
- `src/app/tasks/tasks.module.ts` - Feature module example
- `src/app/shared/card/shared.module.ts` - Shared module pattern
- `src/main.ts` - Module-based bootstrapping
